ğŸ”§ API ENDPOINTS
================

GET /api/v1/activity
- Get recent activities
- Query params: limit (default 100), type, level
- Returns: activities array, count

GET /api/v1/activity/stats
- Get activity statistics
- Returns: total, by_type, by_level, connected_clients

POST /api/v1/activity/clear
- Clear all activities
- Returns: success message

POST /api/v1/activity/start
- Start recording (already auto-started)
- Returns: success message

POST /api/v1/activity/stop
- Stop recording
- Returns: final statistics

GET /api/v1/activity/export
- Export activities as JSON file
- Downloads: activities_YYYY-MM-DD_HH-MM-SS.json

WS /ws/activity
- WebSocket for real-time updates
- Sends initial activities on connect
- Streams new activities as they occur
